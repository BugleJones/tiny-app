# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLS (รก la bit.ly). Users also have the ability to store their unique TinyAppURLs within their personal urls page.

## Final Product

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookieSession

## Getting Started

- Install all dependencies listed (using the ```npm install``` command).
- Run the development web server using the ```node express_server.js``` command.
